# Glycolysis Pathway Simulator

## Overview
This project is a simplified computational model of glycolysis built in Python.

It simulates ATP investment and payoff phases, tracks metabolic changes throughout the pathway, and visualises energy production during glycolysis.

The simulation breaks glycolysis into three main stages and displays changes in glucose, ATP, NADH, and pyruvate throughout each step.

---

## Features

- ATP investment and payoff simulation
- NADH tracking
- Pyruvate production
- ATP visualisation using matplotlib
- Simplified glycolysis pathway flow diagram

---

## Biological Motivation

Glycolysis is one of the most fundamental metabolic pathways in biology.

The pathway breaks down glucose into pyruvate while generating usable cellular energy in the form of ATP and NADH. This project was created to explore how biological pathways can be represented computationally using simplified systems-level modelling.
